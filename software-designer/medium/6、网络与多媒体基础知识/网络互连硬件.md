# 网络互连硬件
## 概念
* 硬件层级
1. 中继器是物理层设备，其作用是对接收的信号进行再生放大，以延长传输的距离。 
2. 网桥是数据链路层设备，可以识别MAC地址，进行帧转发。 
3. 交换机是由硬件构成的多端口网桥，也是一种数据链路层设备。 
4. 路由器是网络层设备，可以识别IP地址，进行数据包的转发。
5. 集线器是物理层设备

* 路由器收到了由多个路由协议转发的、关于某个目标的多条路由，则比较各个路由的管理距离，并采用管理距离小的路由来源提供的路由信息。

* 主机路由的子网掩码是255.255.255.255。网络路由要指明一个子网，所以不可能为全1，默认路由是访问默认网关，而默认网关与本地主机属于同一个子网，其子网掩码也应该与网络路由相同，对静态路由也是同样的道理。

* 把局域网划分为多个不同的VLAN中，允许逻辑地划分网段

* DHCP客户端可从DHCP服务器获得本机IP地址、DNS服务器的地址、DHCP服务器的地址、默认网关的地址等，但没有Web服务器、邮件服务器地址。

* 帧中继（Frame Relay，FR）是为克服X.25交换网的缺陷、提髙传输性能而发展起来的高速分组交换技术。帧中继网络不进行差错和流量控制，并且通过流水方式进行交换，所以比X.25网络的通信开销更少，传输速度更快。 帧中继提供面向连接的虚电路服务，因而比DDN专线更能提高通信线路利用率， 用户负担的通信费用也更低廉。在帧中继网中，用户的信息速率可以在一定的范围内变化，从而既可以适应流式业务，又可以适应突发式业务，这使得帧中继成为远程传输的理想形式。



## 计算题
* IP地址：220.112.179.92，与路由表220.112.177.64/22是否匹配？
> 要确定IP地址 220.112.179.92 是否与路由表条目 220.112.177.64/22 匹配，您需要进行CIDR（无类域间路由）计算。

CIDR 表示一种 IP 地址的编址方法，其中 IP 地址由两部分组成：网络地址和主机地址。CIDR 表示法使用 IP 地址后面的斜线后缀来指定网络的范围。例如，220.112.177.64/22 中的“/22” 表示网络地址的前22位是网络地址，后面是主机地址。

要确定IP地址 220.112.179.92 是否在路由表 220.112.177.64/22 范围内，您可以执行以下步骤：

将IP地址转换为二进制：220.112.179.92 的二进制表示是 11011100.01110000.10110011.01011100。
确定CIDR前缀的网络地址：220.112.177.64/22 中的“/22” 意味着前22位是网络地址，剩下的是主机地址。在二进制表示中，前22位是 11011100.01110000.101100。
比较网络地址：将IP地址的前22位与CIDR前缀的网络地址进行比较。如果它们匹配，那么IP地址就在CIDR范围内。
在这种情况下，220.112.179.92 的前22位（11011100.01110000.101100）与CIDR前缀的网络地址（11011100.01110000.101100）匹配，所以 IP 地址 220.112.179.92 在CIDR 范围内。

因此，IP地址 220.112.179.92 与路由表条目 220.112.177.64/22 匹配。
